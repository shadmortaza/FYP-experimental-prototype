# FYP-experimental-prototype
FYP Prototype 

This repository contains the interim experimental implementations developed for the Final Year Project:

**“Are Neural Networks Really the Most Powerful?
A Comparative Study of ANN and Classical Machine Learning Models”**

## Contents
- `experiment1_har_hmm_vs_ann.ipynb`  
  Sequential classification experiment using the UCI Human Activity Recognition dataset.

- `experiment2_mnist_ann_vs_svm.ipynb`  
  Image classification experiment on MNIST comparing a neural network and a classical baseline.

## Notes
These notebooks are provided as implementation artefacts to demonstrate feasibility at interim stage.  
They are not intended to represent optimised or final experimental results.
